Tugas Pemrograman Web Mobile

Berdasarkan materi yang telah disampaikan sebelumnya, dengan topik Responsive Web:
  • Responsive Web Design (dasar pemahaman)
  • tag: viewport (dalam head-html)
  • Media Query (penggunaan grid dan breakpoints)
  • Responsive Image (sesuai property img)
Kembangkan sebuah aplikasi web dengan ketentuan sebagai berikut:
  • Dikerjakan individu, topik bebas (contoh: Halaman profil)
  • Bahasa yg digunakan minimal HTML5 dan CSS3 (jika diperlukan: Javascript)
  • Diijinkan menggunakan library/framework CSS3 untuk mendukung RWD
    antara lain: bootstrap, materialize, material design lite, bulma, tailwind, ...
  • Minimal 2 halaman web (contoh: halaman Utama dan halaman Detail)
  • Berikan penjelasan dalam kode program berupa komentar, terkait implementasi materi,
    termasuk jika menggunakan library/framework CSS3.
